047=Ce Script est basé sur le programme pour Windows "LigaManager &#39;98 Free" de Frank Hollwitz. <strong>Liga Manager Online</strong> est une version améliorée du programme. Étant basé sur PHP, LMO est utilisabe sur différentes plate-formes, la navigation se fait aisément depuis un navigateur web. Grâce à différentes extentions (Addons) le programme est devenu de plus en plus polyvalent.
048=Avec ce script on peut tout aussi bien éditer des <strong>championnats</strong> de différentes activités sportives, que gérer des <strong>compétitions à élimination directe</strong> (coupes). Un grand nombre d&#39;extensions font de LMO <strong>la solution complète</strong> pour tous ceux qui veulent administrer et publier les résultats de ligues sportives.
